## Qu'est-ce que l'indexation des signets et des pages Web

Contrairement à d'autres outils simples de gestion et de recherche de signets, MemFree génère des "index vectoriels" intelligents pour vos signets et votre contenu web. Cette technologie agit comme la création d'une "carte mentale" pour chaque signet, enregistrant leur signification et leur pertinence. Ainsi, lorsque vous recherchez une question ou une phrase, **MemFree peut trouver les signets et le contenu web les plus correspondants et similaires en fonction de leur sémantique (c'est-à-dire, leur signification réelle)**.

Par exemple, si vous recherchez "meilleur moteur de recherche IA hybride" dans MemFree, il ne se contentera pas de trouver des signets avec ces mots-clés dans le titre, mais aussi ceux liés au contenu de recherche IA, même si le titre ne le mentionne pas explicitement. Cette approche rend les recherches plus intelligentes et efficaces, vous fournissant des informations plus précises et pertinentes.

## Pourquoi indexer des signets et des pages Web

1. Même le plus grand moteur de recherche traditionnel au monde, Google, n'indexe qu'une petite fraction des pages Web dans le monde (estimée à environ 5%).
2. Même si Google les indexe, la probabilité de trouver le contenu exact que vous souhaitez directement sur la première page des résultats de recherche Google est très faible.
3. Pour les signets et les pages Web que vous indexez spécifiquement dans MemFree, **MemFree les priorisera lors des recherches, en utilisant l'indexation vectorielle pour renvoyer le contenu le plus similaire, et fournira finalement la meilleure réponse grâce à son modèle AI**.
4. Pour les pages Web que vous trouvez précieuses ou intéressantes, il vous suffit de laisser MemFree les indexer d'un simple clic. MemFree deviendra votre second cerveau, éliminant le besoin de vous souvenir des détails. Plus tard, **lorsque vous souhaiterez récupérer des informations des pages Web indexées, MemFree vous fournira les réponses précises dont vous avez besoin**.

En tirant parti de cette approche, MemFree ne se contente pas de gérer et de rechercher intelligemment et efficacement vos signets et vos pages Web, mais garantit également que vous pouvez toujours localiser rapidement les informations dont vous avez besoin, qu'elles soient cachées quelque part en ligne ou stockées dans votre collection personnelle.

## Comment indexer des signets et des pages Web

### Comment indexer des signets

Veuillez consulter le [Guide de l'utilisateur de l'extension Chrome MemFree](/docs/extension-user-guide) pour des instructions détaillées sur la façon d'indexer des signets à l'aide de l'extension Chrome MemFree.

### Comment indexer des pages Web

1. Indexez la page Web en cliquant sur le bouton "Indexer" dans l'extension Chrome MemFree.

    className="mt-4 overflow-hidden rounded-lg border"

Le bouton d'indexation est suspendu dans le coin inférieur droit de n'importe quelle page Web après avoir installé l'extension Chrome MemFree.

**Pour les pages Web qui nécessitent une connexion, vous devez utiliser l'extension Chrome pour les indexer.**

2. Indexez la page Web dans l'application web MemFree en collant l'URL et en cliquant sur le bouton "Indexer".
