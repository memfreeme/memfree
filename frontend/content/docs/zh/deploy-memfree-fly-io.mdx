## 如何在 Fly.io 上部署 MemFree 向量服务

<Steps>

### 获取 Fly.io 账户（可选）

-   前往 [Fly.io](https://fly.io) 并创建一个账户。

### 安装 Flyctl（可选）

-   按照 [这里](https://fly.io/docs/flyctl/install/) 的说明安装 `flyctl`。

### 从 GitHub 克隆 MemFree 代码

```js
git clone https://github.com/memfreeme/memfree.git
```

### 部署 MemFree 向量服务

```js
cd vector

cp fly.example.toml fly.toml

将 fly.toml 中的 APP_NAME 更改为您的应用名称

fly deploy

```

</Steps>
