## ブックマークとウェブページのインデックス化とは

他の単純なブックマーク管理および検索ツールとは異なり、MemFreeはブックマークやウェブコンテンツのためのインテリジェントな「ベクトルインデックス」を生成します。この技術は、各ブックマークの「マインドマップ」を作成するように機能し、その意味と関連性を記録します。したがって、質問や文を検索すると、**MemFreeは意味（実際の意味）に基づいて最も一致するブックマークやウェブコンテンツを見つけることができます**。

例えば、MemFreeで「最高のハイブリッドAI検索エンジン」を検索すると、そのキーワードがタイトルに含まれるブックマークだけでなく、明示的にタイトルには含まれていなくてもAI検索コンテンツに関連するものも見つけます。このアプローチにより、検索がよりスマートかつ効率的になり、より正確で関連性の高い情報を提供します。

## なぜブックマークとウェブページをインデックス化するのか

1. 世界最大の伝統的な検索エンジンであるGoogleでさえ、世界中のウェブページのごく一部（推定約5%）しかインデックス化していません。
2. Googleがインデックス化しても、Googleの検索結果の最初のページに求めている正確なコンテンツを見つける可能性は非常に低いです。
3. MemFreeで特にインデックスしたブックマークやウェブページについては、**MemFreeは検索時にこれらを優先し、ベクトルインデックスを利用して最も類似のコンテンツを返し、最終的にAIモデルを通じて最良の回答を提供します**。
4. 価値のあるウェブページや興味深いウェブページを見つけた場合、MemFreeにワンクリックでインデックス化させることができます。MemFreeはあなたのセカンドブレインとなり、特定のことを記憶する必要がなくなります。後で、**インデックス化されたウェブページから情報を取得したいときは、MemFreeが必要な正確な回答を提供します**。

このアプローチを活用することで、MemFreeはあなたのブックマークやウェブページをスマートかつ効率的に管理・検索するだけでなく、オンラインのどこかに隠れている情報やあなたの個人コレクションに保存されている情報を迅速に見つけることができるようにします。

## ブックマークとウェブページのインデックス化方法

### ブックマークのインデックス化方法

MemFree Chrome拡張機能を使用してブックマークをインデックス化する方法については、[MemFree Chrome拡張機能ユーザーガイド](/docs/extension-user-guide)を参照してください。

### ウェブページのインデックス化方法

1. MemFree Chrome拡張機能の「インデックス」ボタンをクリックしてウェブページをインデックス化します。

<Image
    className="mt-4 overflow-hidden rounded-lg border"
    src="/images/changelog/extension-index.webp"
    width="800"
    height="450"
    alt="memfree chrome extension index web page"
/>

インストール後、任意のウェブページの右下隅にインデックスボタンが表示されます。

**ログインが必要なウェブページについては、Chrome拡張機能を使用してインデックス化してください。**

2. MemFreeウェブアプリでURLを貼り付けて「インデックス」ボタンをクリックしてウェブページをインデックス化します。
