---
title: MemFree als Ihre Suchmaschine einstellen
description: So stellen Sie MemFree als Ihre standardmäßige Suchmaschine im Browser ein
---

<Steps>

### Öffnen Sie die Browsereinstellungen

Geben Sie die URL "chrome://settings/searchEngines" in die URL-/Navigationsleiste ein.

### Fügen Sie MemFree als Suchmaschine hinzu

<img className="mt-4 overflow-hidden rounded-lg border" src="/images/doc/add-search-engine1.png" alt="memfree zur Browsersuchmaschine hinzufügen 1" />

<img className="mt-4 overflow-hidden rounded-lg border" src="/images/doc/add-search-engine2.png" alt="memfree zur Browsersuchmaschine hinzufügen 2" />

-   Der Name der Suchmaschine: MemFree
-   Die Abkürzung: memfree
-   Die URL mit %s anstelle der Abfrage: https://www.memfree.me/search?q=%s

### Stellen Sie MemFree als standardmäßige Suchmaschine ein

<img
    className="mt-4 overflow-hidden rounded-lg border"
    src="/images/doc/default-search-engine.png"
    alt="memfree als Standard-Suchmaschine im Browser einstellen"
/>
</Steps>

Jetzt können Sie MemFree direkt aus der Suchleiste Ihres Browsers verwenden.
