---
title: Set MemFree as your search engine
description: How to set MemFree as your default search engine in browser
---

<Steps>

### Open the browser settings

Enter the URL "chrome://settings/searchEngines" in the URL/navigation bar.

### Add MemFree as a search engine

<img className="mt-4 overflow-hidden rounded-lg border" src="/images/doc/add-search-engine1.png" alt="add memfree to browser search engine 1" />

<img className="mt-4 overflow-hidden rounded-lg border" src="/images/doc/add-search-engine2.png" alt="add memfree to browser search engine 2" />

-   The search engine name: MemFree
-   The Shortcut: memfree
-   The URL with %s in place of query: https://www.memfree.me/search?q=%s

### Set MemFree as the default search engine

<img
    className="mt-4 overflow-hidden rounded-lg border"
    src="/images/doc/default-search-engine.png"
    alt="set memfree to browser default search engine"
/>
</Steps>

Now, you can use MemFree directly from your browser's search bar.
