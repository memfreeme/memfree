## كيفية نشر خدمة MemFree Vector على Fly.io

<Steps>

### الحصول على حساب Fly.io (اختياري)

-   اذهب إلى [Fly.io](https://fly.io) وأنشئ حسابًا.

### تثبيت Flyctl (اختياري)

-   قم بتثبيت `flyctl` عن طريق اتباع التعليمات [هنا](https://fly.io/docs/flyctl/install/).

### استنساخ كود MemFree من GitHub

```js
git clone https://github.com/memfreeme/memfree.git
```

### نشر خدمة MemFree Vector

```js
cd vector

cp fly.example.toml fly.toml

غير اسم APP_NAME في fly.toml إلى اسم تطبيقك

fly deploy



```

</Steps>
